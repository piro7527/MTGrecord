# キッチンリフォーム相談 打合せまとめ

> **場所**：ショールーム（展示を見ながらの打合せ）  
> **状況**：施工業者はすでに決定済み。来週中に採寸の予定。キッチンメーカー担当者との仕様確認。

---

## 1. 天板（カウンタートップ）

| 項目 | 内容                                                        |
| ---- | ----------------------------------------------------------- |
| 素材 | ステンレス → **ファイナルクォーツ**に変更（傷・汚れに強い） |

---

## 2. シンクボール

| 項目   | 内容                                                         |
| ------ | ------------------------------------------------------------ |
| 素材   | **ステンレス**（従来どおり）                                 |
| サイズ | 現行より少し大きめ（約108cm程度）を確認中                    |
| 備考   | クォーツ天板との組み合わせで取り付け可能か業者に再確認が必要 |

---

## 3. 水栓

| 項目       | 内容                                                           |
| ---------- | -------------------------------------------------------------- |
| 手配方法   | IKEAで気に入ったものを**別途自己手配**する方向                 |
| 取り付け   | 施工業者に依頼。IKEAの製品が取り付け対応可能か業者に確認が必要 |
| 見積もり上 | 「水栓なし」として処理                                         |

---

## 4. キッチン本体シリーズの選定

| シリーズ                         | 特徴                                                        | 価格感   |
| -------------------------------- | ----------------------------------------------------------- | -------- |
| **レミュー**（本命）             | 色・デザインが気に入っている。内引き出し5段など選択肢が豊富 | 標準     |
| **トレッシャ**                   | 引出し3段（標準）、カラーはシルバーのみ                     | やや高め |
| **リフィット**（新シリーズ候補） | 60cm幅の食洗機対応。4月1日以降に価格変更あり                | 要確認   |

> まず **レミュー** で見積もりを出し、後からシリーズを比較検討する方針

---

## 5. 扉カラーの検討

| 候補カラー                 | 印象・コメント                               |
| -------------------------- | -------------------------------------------- |
| **パールホワイト**（有力） | キラキラしたパール感。グループ2（中価格帯）  |
| ピュアアイボリ             | 温かみのあるアイボリ、赤みが少なくベージュ系 |
| グレー                     | 清潔感あり、落ち着いた選択肢                 |
| ホワイト（無地）           | シンプルで定番。マグネット小物も使えて遊べる |

> ⚠️ **パールホワイト**に変更した場合、金額が**約10万円弱**上がる見込み。試算が必要。

---

## 6. 壁パネル

- キッチン扉の色に合わせてパネルも統一するのが望ましい
- タイルが理想だが、メーカー在庫では対応が難しく業者手配になる
- **後からパネルを穴あけ追加する場合は防錆加工ができない**ため、施工時に一緒に決める必要がある

---

## 7. 内部収納オプション

| オプション                               | 採用方針                                            |
| ---------------------------------------- | --------------------------------------------------- |
| 上段内引き出し（マグネット白仕切り付き） | **標準付属**。そのまま採用                          |
| 中段・下段の仕切り類（四角仕切り等）     | **不要**（後から購入可能なので今回は外す）          |
| まな板・作業スペース延長板               | **不要**（たまにしか使わないため外す）              |
| 鍋スタンド（縦置き）                     | **鍋置きスペースのみ2セット**採用、その他仕切りなし |
| カトラリートレー等                       | 今回は外す（市販のボックスで対応予定）              |

---

## 8. 価格・スケジュール

| 項目               | 内容                                                  |
| ------------------ | ----------------------------------------------------- |
| 現在の概算         | **100万円以下**の見込み（レミュー、現行カラーの場合） |
| カラー変更の影響   | パールホワイト採用で**約10万円弱**追加                |
| **価格改定の期限** | ⚠️ **3月25日まで**が現行価格。以降は価格変動あり       |
| 採寸               | **来週中**に業者が自宅に採寸に来る予定                |

---

## 9. 今後のアクション

| 担当             | タスク                                                                                                            |
| ---------------- | ----------------------------------------------------------------------------------------------------------------- |
| **客側**         | ①扉カラーの最終決定（パールホワイトか否か）②水栓をIKEAで選定し、業者へ取付可否確認③壁パネルの要否・デザインを決定 |
| **メーカー担当** | ①収納オプションを整理した最新見積もりを作成②パールホワイト採用の場合の試算を追加③業者と連携して採寸・施工調整     |

---

## まとめ

- **天板**：クォーツに変更確定方向
- **シンク**：ステンレス継続
- **シリーズ**：レミューで見積もりを出し比較検討
- **扉色**：パールホワイトが有力だが**+約10万円**のため最終確認が必要
- **期限**：**3月25日**までに発注が必要（以降価格改定）
