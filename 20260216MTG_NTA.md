# 海外研修プログラム検討会議 議事録まとめ

2026.02.16.
日本旅行山田さん、青山、林

## 1. 概要
オーストラリア（グリフィス大学）への海外研修プログラムに関する打ち合わせ。日程、航空会社の選定、費用、および今後のスケジュールについて議論が行われた。

## 2. 日程
- **基本パターン**: 2週間のプログラム。土曜日日本出発 → 日曜日現地到着 → 月曜日から研修開始。
- **候補日程**:
    1. **8月8日出発**: 回答済み。
    2. **8月1日出発**: 見積もり・空き状況確認中。

## 3. 航空会社の比較
| 項目                 | カンタス航空 (QF)          | キャセイパシフィック航空 (CX)                               | 備考                               |
| :------------------- | :------------------------- | :---------------------------------------------------------- | :--------------------------------- |
| **ルート**           | 成田 ⇔ ブリスベン (直行便) | 関空 ⇔ 香港乗り継ぎ ⇔ ブリスベン                            | カンタスは伊丹→羽田/成田移動あり   |
| **利便性**           | 高い (直行便)              | 低い (乗り継ぎあり、帰路は香港での待機時間長、または機中泊) |                                    |
| **費用**             | 高い (約67万円)            | カンタスより約2.2万円安い                                   | キャセイは燃油サーチャージ別途必要 |
| **シンガポール航空** | 不可                       | 不可                                                        | 繁忙期のため団体予約不可           |

## 4. 費用・為替について
- **現状**: 昨年比で費用増。主な要因は航空券代の高騰と円安（豪ドル：昨年約105円 → 現在約110円）。
- **為替リスク対策**:
    - レートを確定させてドルを一括購入する案（先物予約）。
    - メリット: 費用が確定し、追加請求のリスクがない。
    - デメリット: その後レートが下がっても返金は不可。現状より2万円程度下がる見込みで計算可能。
- **個人手配**: 学生が航空券を個人手配すれば安くなる可能性があるが、管理が複雑になるため要検討。

## 5. 今後のアクション
1. **見積もり提示**: 8月1日出発分と8月8日出発分の両方の見積もり（レート調整後）を旅行会社から提示する（2-3日以内）。
2. **決定**: 学内で検討し、日程と航空会社を決定する。
3. **告知**: 3月下旬〜4月上旬を目処にポスター等の募集要項を完成させ、学生（新入生含む）へ告知する。

## 6. その他検討事項（将来的な展望）
- **他国での研修**: 台湾や韓国など、アジア圏での短期研修（費用半額程度）の可能性についても話題に上がった。
- **病院見学**: 現在のパッケージには病院見学費用が含まれている。
