# キッチンリフォーム相談まとめ

## 1. 概要
- **トピック**: キッチンリフォーム相談（タカラスタンダード）
- **顧客状況**: 築約20年のキッチンのリフォームを検討中。
- **背景**: 4月1日の価格改定前に注文したい（3月末までの契約が必要）。

## 2. 現状と課題
- **現在のキッチン**: 約20年使用、幅255cm。
- **課題**:
  - 下台の収納からにおいがする（カビ？）。
  - 食洗機が小さすぎて使っていない（物置状態）。
  - 収納が開き戸タイプで使いにくい（奥のものが取り出しにくい）。
  - ステンレス天板の汚れが落ちにくい。
- **制約事項**:
  - 正面の窓により吊戸棚の高さに制限がある。
  - 換気扇のダクトが横出し（施工可能と確認済み）。

## 3. 顧客の要望と要件
- **メーカー**: タカラスタンダード（ホーローに興味あり）。
- **検討シリーズ**:
  - **レミュー (Lemure)**: 見積り済み（上位モデル）。
  - **トレーシア (Treasia)**: 当初検討していた（中級モデル）。
- **主な仕様**:
  - **天板**: ステンレス希望（熱い鍋を置きたい、気兼ねなく使いたい）。
  - **キャビネット**: ホーロー（耐久性、清掃性）。
  - **食洗機**: 深型など、容量の大きいものを希望。
  - **熱源**: ガス（現状）かIHかで迷っている（IHにするなら200V工事とブレーカー確認が必要）。
  - **キッチンパネル**: ホーローパネル（掃除が楽）。

## 4. 予算と見積もり
- **想定費用**: 商品代 約150万円 + 工事費 約15万円 + 施工業者費用。
  - 顧客の予算感は約200万円だったが、見積もり（レミュー）は150万円弱。
  - トレーシアに変更し、天板をステンレスにすれば30-40万円抑えられる可能性あり。
- **備考**: 4月1日から価格改定（値上げ）があるため、現行価格での契約には期限がある。

## 5. スケジュール
- **期限**: 3月31日までの注文で現行価格適用。
- **工事時期**: 4月〜6月頃（顧客は5月が忙しい可能性、6月頃を希望）。
- **流れ**:
  1. ショールーム見学（完了）。
  2. 業者による現地調査（未定、これから依頼）。
  3. 正式見積もり（2-3週間）。
  4. 契約・発注（3月末まで）。
  5. 施工（最短3日間）。

## 6. アクションアイテム
- **スタッフ**:
  - プラン・見積もりをメールで送付（当日中〜翌日）。
- **顧客**:
  - 施工業者の選定（ロイヤルホームセンターまたは紹介業者）。
  - ブレーカー容量の確認（IH導入検討のため）。
  - 見積もり内容の確認と仕様調整（色、シリーズなど）。
  - 次回の打ち合わせ予約また連絡。
